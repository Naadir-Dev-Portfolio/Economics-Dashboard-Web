# Economics-Dashboard-Web

Coming soon — a fully interactive, browser-based economics dashboard built with JavaScript

Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

## Overview

A comprehensive economics and macro data visualization platform. Features interactive charts for yield curves, housing data, employment metrics, inflation trends, and more. Pulls live data from public economic APIs.

## Features

- Interactive yield curve visualization
- Housing market metrics and trends
- Employment and labor statistics
- Inflation analysis and projections
- Real-time macro data from public APIs
- Dark mode responsive design

## Tech Stack

JavaScript · HTML5 · CSS3 · D3.js/Chart.js · Public APIs
